## Installation

1. Install Expo CLI

```js
npm install --global expo-cli
```

2. After cloning this repo, add the following to `.env` file:

```shell
SUPABASE_URL=url
SUPABASE_KEY=key
```

3. Install deps

```js
yarn
```

4. Start Expo server:

```js
expo start
```
